# Else-if

The construction:
```c
if (expression)
    statement;
else if (expression)
    statement;
else if (expression)
    statement;
else if (expression)
    statement;
else
    statement;
```
is very common. The sequence of `if` statements is the most general way of writing a multi-way decision. The _expressions_ are evaluated in order; if an _expression_ is true, the statement associated with it is executed, and this terminates the whole chain.
The last `else` handles the _"none of the above"_ case.
