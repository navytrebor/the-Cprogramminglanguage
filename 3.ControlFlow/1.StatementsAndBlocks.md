# Statements and Blocks

An expression like `x = 0` becomes a **statement** when it is followed by a semicolon, as in:
 ```c
 x = 0;
 ```
 In C, the semicolon is a statement terminator.

 Braces { and } are used to group declarations and statements together into a _compound statement_, or _block_, so that they are syntactically equivalent to a single statement.
 There is no semicolon after the right brace that ends a block.
