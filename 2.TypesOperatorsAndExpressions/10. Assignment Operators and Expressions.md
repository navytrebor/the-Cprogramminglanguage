# Assignment Operators and Expressions

An expressions such as:
```c
i = i + 2
```
can be written in compressed form:
```c
i += 2
```
The operator `+=`is called **assignment operator**.

Most binary operators have a corresponding assignment operator *op=* where *op* is one of
`+ - * / % << >> & ^ |`
if *expr1* and *expr2* are expressions, then:
```c
expr1 op= expr2
```
is equivalent to
```c
expr1 = (expr1) op (expr2)
```
**Example with the bitcount function**

The `bitcount` function counts the number of 1-bits in its integer argument.
```c
/* bitcount: count 1 bits in x */
int bitcount(unsigned x)
{
    int b;
    for (b = 0; x != 0; x >>= 1)
        if (x and 01)
          b++;
    return b;
}
```
Declaring the argument x to be an unsigned ensures that when it is right-shifted, vacated bits will be filled with zeros, not sign bits, regardless of the machine the program is run on.

In all such expressions, the type of an assignment expression is the type of its left operand, and the value is the value after the assignment.
